Proyek 2 Data Supervised dan Unsupervised Learning 🎯#

Objective 📝#

  • Perkenalan

  • Supervised

  • Unsupervised

Perkenalan#

  • Data dibagi menjadi 3 yaitu supervised, unsupervised, reinforcement learning

  • Ilmuwan AI Yann Lecun memperkenalkan cake analogy

  • Tipe hasil data berjenis continue dan discrete

Data Kontinyu#

  • Data yang dapat diukur jaraknya

  • Data yang memiliki jangka atau interval

  • Contoh kasus “Prediksi harga rumah”

  • Hadil model memprediksi jangka hargan \( Rp.499.999.000 \ ∼ \ Rp. 500.100.000 \),

  • padahal harga seharusnya adalah Rp. 500.000.000. Perbedaan ini tak jadi masalah sebab hasil prediksi dalam regresi linear akan berada dalam range bilangan numerik tertentu yang mendekati nilai seharusnya.

Data Diskrit#

  • Data yang dapat dihitung

  • Data yang memiliki nilai berbeda dari perhitungan

  • Contoh kasus “Penentuan E-mail apakah termasuk spam atau tidak”

  • Apakah email dapat diklasifikasikan sebagai spam atau bukan spam?

  • Email tidak memiliki kontinuitas antara email spam dan bukan spam.

  • Email spam tidak sama dengan bukan spam

  • Email tidak memiliki kategori antara email spam dan bukan spam.

Cake Analogy#

“If intelligence is a cake, the bulk of the cake is unsupervised learning, the icing on the cake is supervised learning, and the cherry on the cake is reinforcement learning (RL).”

  • Jika sistem kecerdasan adalah sebuah kue

  • Unsupervised Learning adalah kuenya

  • Supervised Learning adalah lapisan gula pada kue

  • Reinforcement Learning adalah buah cherry pada kue

  • Unsupervised Learning \( 100.000 ∼ 1000.000 \) lebih banyak dari Supervised Learning

  • Supervised Learning \( 1.000 ∼ 10.000 \) lebih banyak dari Reinforcement Learning

  • Reinforcement Learning \( 10 ∼ 100 \)

Supervised 🧮💻#

  • Supervised adalah data yang memiliki label

  • Label berfungsi untuk mengenali sebuah data

  • Label membantu proses klasifikasi beberapa data`

Algoritma supervised learning#

Linear Regression#
Logistic Regression#
Classification#
Decision Trees#
Support Vector Machines#
Neural Networks#

Supervised : Classification#

Binary Classification#

  • Klasifikasi dataset yang hanya terdiri dari 2 kelas

Multiclass Classification#

  • Klasifikasi dataset yang memiliki lebih dari 2 kelas

  • Contoh dataset bunga iris

Decision Tree#

  • Salah satu algoritma yang powerfull

  • Decision tree juga merupakan komponen dasar algoritma Random Forest

import numpy as np
import pandas as pd
from sklearn.datasets import load_iris
from sklearn import tree
 
# membaca file iris.csv
data = load_iris()
iris = pd.DataFrame(data= np.c_[data['data'], data['target']],
                     columns= data['feature_names'] + ['target_names'])

iris['target_names'] = iris['target_names'].replace(to_replace= [0, 1, 2], value = ['setosa', 'versicolor', 'virginica'])

# memisahkan atribut dan label
X = iris[['sepal length (cm)', 'sepal width (cm)', 'petal length (cm)', 'petal width (cm)']]
y = iris['target_names']

# Membagi dataset menjadi data latih & data uji
from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.1, random_state=123)

from sklearn.tree import DecisionTreeClassifier
 
# membuat model Decision Tree
tree_model = DecisionTreeClassifier() 
 
# melakukan pelatihan model terhadap data
tree_model = tree_model.fit(X_train, y_train)

# Evaluasi Model
from sklearn.metrics import accuracy_score

y_pred = tree_model.predict(X_test)

acc_secore = round(accuracy_score(y_pred, y_test), 3)

print('Accuracy: ', acc_secore)
